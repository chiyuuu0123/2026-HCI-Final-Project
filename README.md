# MindStudy：多模态智能学习伙伴

MindStudy 是一款面向大学生课程学习的桌面端智能学习软件。它的重点不是做理论研究，而是做出一个界面精美、流程顺畅、功能实用的学习助手，让学生能够把 PPT、PDF 等课程资料变成可理解、可练习、可追踪的学习内容。

软件将 AI 课程问答、知识图谱、自动测验、错题追踪、学习进度、情绪状态提醒、背景音乐和手势控制整合在一个完整的学习工作台中，目标是让学习过程更清晰、更轻松、更有掌控感。

## 产品定位

MindStudy 面向需要高效复习课程资料的大学生，尤其适合期末复习、课堂资料整理、课程重点梳理和自测练习场景。

它希望解决这些实际问题：

- 资料太多，不知道从哪里开始看。
- PPT 和 PDF 内容零散，难以形成完整知识结构。
- 学习时遇到概念不懂，需要快速追问。
- 复习缺少练习题和错题反馈。
- 学习时间久了容易疲劳、分心或焦虑。
- 操作频繁依赖鼠标键盘，不够自然和沉浸。

## 核心功能

### 资料上传与解析

用户可以上传课程 PPT、PDF 或讲义文件。系统自动提取正文、标题、章节结构、关键词和重点概念，并将资料整理成可浏览的学习内容。

### 智能摘要与知识图谱

系统根据课程内容生成章节摘要和知识图谱，将零散知识点组织成清晰的结构。用户可以点击知识点查看解释、例子、相关原文和掌握状态。

### 课程 AI 问答

用户可以围绕上传资料提问，例如“这一章重点是什么”“这个概念怎么理解”“这页 PPT 可能怎么考”。系统基于课程资料回答，并尽量给出来源和解释。

### 自动测验与错题本

系统可以根据章节、知识点或难度生成练习题，支持选择题、判断题、简答题和概念匹配题。答题后自动记录错题，并推荐需要复习的内容。

### 学习进度面板

软件提供学习时长、章节完成度、测验正确率、薄弱知识点和复习建议，让学生知道自己学到哪里、哪里还不熟。

### 情绪状态提醒

系统通过摄像头检测学习者的疲劳、困惑、焦虑或注意力下降状态，并给出轻量提醒，例如建议短暂休息、切换练习模式或降低学习强度。

### 背景音乐推荐

软件根据学习状态推荐低干扰背景音乐、白噪音、自然声或舒缓音乐，帮助用户维持专注或缓解疲劳。

### 手势控制

软件支持用摄像头识别基础手势，实现翻页、暂停、确认答案、截图收藏和切换题目等无接触操作。

| 手势 | 功能 |
| --- | --- |
| 张手 | 下一页 / 展开知识点 |
| 握拳 | 暂停学习 / 暂停音乐 |
| 点赞 | 确认答案 |
| 比 V | 截图 / 收藏当前内容 |
| 左右挥手 | 上一题 / 下一题 |

## 典型使用流程

1. 打开 MindStudy，进入学习工作台。
2. 上传课程 PPT 或 PDF。
3. 系统解析资料并生成摘要、关键词和知识图谱。
4. 用户浏览知识图谱，点击知识点查看解释和原文。
5. 用户向 AI 提问课程相关问题。
6. 系统生成练习题，用户进行自测。
7. 软件记录错题、正确率和薄弱知识点。
8. 学习过程中检测到疲劳或困惑时，系统给出提醒。
9. 用户可以播放推荐音乐，也可以通过手势切换页面或题目。
10. 最后查看学习报告和个性化复习建议。

## 软件界面规划

MindStudy 的界面目标是清爽、现代、易用，适合作为一个真正可演示的软件原型。

主要页面包括：

- **首页 / 工作台**：展示最近资料、今日学习进度、待复习内容和快速入口。
- **资料阅读页**：左侧目录，中间资料预览，右侧 AI 助手和笔记。
- **知识图谱页**：可视化展示课程知识结构，支持节点筛选和状态标记。
- **AI 问答页**：支持连续追问、来源展示和保存为笔记。
- **测验页**：展示题目、答案、解析和错题记录。
- **学习报告页**：展示学习时长、正确率、薄弱知识点和复习建议。
- **音乐与状态栏**：展示当前学习状态、音乐推荐和摄像头识别状态。

## 项目亮点

- 将课程资料整理、问答、练习和复习反馈放进一个完整学习流程。
- 用知识图谱把 PPT / PDF 中的内容变得结构化、可视化。
- 支持基于资料的 AI 问答，而不是泛泛聊天。
- 自动生成练习题和错题本，真正服务复习。
- 通过情绪提醒和音乐推荐改善学习体验。
- 通过手势控制增加自然、沉浸的交互方式。
- 注重界面完成度和课堂演示效果，目标是做出精美实用的软件原型。
