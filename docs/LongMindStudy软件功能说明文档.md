# LongMindStudy-龙龙多模态智能学习伙伴软件功能说明文档

## 一、软件概述

LongMindStudy 是一款面向课程学习与期末复习的桌面端智能学习软件。软件把课程资料阅读、资料解析、知识库问答、知识图谱、思维导图、自动测验、错题复习、学习报告、情绪状态识别、音乐推荐、语音控制和桌宠陪伴整合在同一个学习工作台中。用户在一个软件内完成资料导入、资料理解、提问答疑、自测练习、进度追踪和状态调节，学习过程形成连续闭环。

软件以课程为基本组织单位。每一门课程拥有独立的资料库、RAG 知识库、知识图谱、思维导图、测验记录、错题记录和学习报告。用户切换课程时，软件同步切换该课程对应的资料、图谱、测验与报告内容，使不同课程的学习进度保持清晰分离。

LongMindStudy 的主要使用对象是需要集中复习课程资料的学生。软件适合处理课堂 PDF、Markdown 笔记、复习提纲、实验说明和其他文本型学习资料。用户导入资料后，软件把原本静态的文档转化为可阅读、可提问、可检索、可练习、可回顾的学习内容。

## 二、整体架构

LongMindStudy 采用 Electron 桌面应用架构。软件由桌面窗口层、前端交互层、本地文件能力层、AI 能力层、图数据库能力层、多模态识别层和本地持久化层组成。

桌面窗口层负责创建主学习窗口和龙龙桌宠窗口。主窗口承载完整学习工作台，桌宠窗口以悬浮形态出现在桌面上，用于学习提醒、陪伴聊天、好感互动和快速唤醒主窗口。

前端交互层负责页面导航、课程切换、资料阅读、图谱展示、测验答题、学习报告、日程管理、计算器、状态识别界面和音乐播放器。界面使用原生 HTML、CSS 与 JavaScript 构建，整体布局采用左侧导航栏加右侧工作区的结构，主工作区根据功能切换不同视图。

本地文件能力层负责打开本机资料、读取文件内容、保存 Markdown、导出 PDF 副本、保存 AI 阅读内容、读取音乐素材和保存用户配置。文件读写通过 Electron 的安全桥接层完成，前端通过统一接口调用桌面端能力。

AI 能力层负责 Qwen / DashScope 接口调用、课程问答、资料摘要、RAG 知识库回答、简答题语义评分、代码助手、语音识别和多模态输入处理。API Key 支持通过软件内设置保存到本机应用数据目录，也支持通过环境变量读取。

图数据库能力层负责 Neo4j 连接、知识图谱生成结果保存、课程图谱读取、节点邻居查询、节点搜索和路径查询。软件把课程知识点和概念关系写入本地图数据库，再在前端渲染为可交互知识网络。

多模态识别层负责摄像头画面采样、MediaPipe 手势识别、面部状态识别、专注分计算、手势鼠标控制和音乐推荐状态输入。该层把用户学习过程中的状态变化转化为界面反馈、音乐策略和学习报告数据。

本地持久化层保存课程资料库、学习模块、RAG 知识库、学习计划、日历事件、学习计时、龙龙记忆、龙龙好感、桌宠位置、AI Key 配置和阅读布局设置。用户再次打开软件时，课程、资料记录、学习状态和助手互动保持连续。

## 三、主界面与导航结构

软件主界面左侧为垂直导航栏，右侧为当前功能页面。导航栏提供以下入口：

- 学习工作台
- 资料阅读
- 知识库问答
- 阿龙 Coding
- 科学计算器
- 学习计划
- 知识图谱
- 思维导图
- 自动测验
- 学习报告
- 状态与音乐

顶部区域显示当前视图标题、语音控制按钮、摄像头状态按钮和资料导入按钮。用户可以在任意页面通过顶部导入按钮添加课程资料，也可以通过课程选择器切换当前课程。

软件支持全局缩放。用户按住 `Ctrl` 并滚动鼠标滚轮可调整界面比例，`Ctrl + 0` 恢复默认比例。缩放设置保存在本机，适合不同屏幕尺寸和演示场景。

软件支持语音控制面板。用户点击顶部语音控制按钮后，语音控制进入持续监听状态。软件只对可执行的控制指令做出响应，例如切换页面、打开资料、跳转 PDF 页码、学习资料、生成知识图谱、开启摄像头和打开龙龙聊天。普通闲聊或与控制无关的语音会被忽略，使语音模式更接近“控制入口”而不是普通输入框。

## 四、课程与资料库

课程是软件管理学习内容的最高层级。每门课程包含课程名称、课程说明、资料列表、当前激活资料、学习图谱、思维导图、RAG 知识库、测验数据、错题数据和学习报告数据。

用户可以在课程选择器中新建课程、编辑课程名称和课程说明，也可以在多个课程之间切换。切换课程后，资料阅读页、RAG 问答页、知识图谱页、思维导图页、自动测验页和学习报告页都会显示该课程对应的数据。

资料库支持导入 PDF 和 Markdown。导入时，软件记录文件名称、扩展名、大小、本机路径、更新时间和文件内容。PDF 资料以阅读器方式打开，Markdown 资料以预览与编辑方式打开。资料记录保存在课程中，用户下次打开软件后可以继续从资料库进入对应文件。

资料库中的每个文件可以从课程列表中移除。移除操作只改变软件课程资料库中的记录，学习流程围绕当前课程资料重新组织。

## 五、学习工作台

学习工作台是软件的首页。页面展示当前课程状态、资料导入入口、学习计时、日历事件和近期学习提示。工作台帮助用户快速判断当前课程是否已经导入资料、今天学习了多久、近期有哪些重要事项。

工作台包含日历模块。用户可以在日历中切换月份、选择日期、添加事件并记录备注。事件类型包括考试、DDL、答辩、面试和自定义学习事项。事件可以设置重要程度，软件在日历和事件列表中展示日期、类型、标题和备注。

工作台包含当前课程资料卡片。资料卡片展示当前课程最新或正在使用的资料名称、资料类型、文件大小和资料库状态。用户点击导入按钮后选择本地 PDF 或 Markdown，资料进入当前课程资料库。

工作台包含学习计时。软件打开后自动维护今日学习时长，计时结果同步到顶部、工作台、桌宠和学习计划日历热力信息。学习计时也用于龙龙龙币结算和阶段学习回顾。

## 六、资料阅读

资料阅读页是课程资料的主学习区域。页面采用三栏结构：左侧资料库与章节列表，中间文档阅读区，右侧智能侧栏。三栏之间带有可拖拽分隔条，用户可以根据屏幕空间调整资料栏和智能栏宽度。

PDF 阅读器支持页面渲染、页码切换、上一页与下一页操作、当前页文字提取、OCR 识别、批注保存、画笔批注、图片插入、添加空白页、删除当前页、导出带编辑内容的 PDF 副本和全屏阅读。用户在 PDF 页面中切换页码后，软件自动尝试读取当前页文字，并把当前页文字带入右侧 AI 阅读窗口。

PDF 的文字提取同时支持文本层和图片型页面。带文本层的 PDF 会直接提取页面文字；图片型页面通过把页面渲染为图像后调用 OCR 识别。OCR 支持中英文识别，识别后的文本会合并进资料解析结果，用于 AI 阅读、RAG 学习和知识库问答。

PDF 页段工具支持输入起始页和结束页。用户可以把指定页段导出为子 PDF，也可以只让知识库学习某个页段。该能力适合处理较大的教材、扫描版 PDF 和只关注某几页内容的复习场景。

Markdown 阅读器支持正文预览、编辑、保存到原文件和另存为副本。预览器支持标题、段落、列表、表格、代码块、链接和数学公式显示。AI 对话输出与 Markdown 预览都使用更接近人类书写方式的公式呈现，行内公式以自然数学形式显示，较复杂公式以独立公式块显示。

智能侧栏包含 AI 阅读窗口。用户可以查看当前页或当前文档提取出的文字，也可以让软件分析内容、提炼重点、给出复习提示或把英文内容翻译为中文。AI 阅读的原文框和输出框支持弹出为浮动窗口，便于复制、保存和阅读较长内容。

全屏阅读模式把资料阅读区域扩展到更沉浸的三栏布局。用户在全屏中继续切换资料、阅读 PDF、编辑 Markdown、查看 AI 阅读内容和保存笔记。退出全屏后，阅读位置和布局设置保持同步。

## 七、知识库问答助手

知识库问答页提供 RAG 知识问答能力。用户先让软件学习当前文件、学习整个课程或学习 PDF 指定页段，软件把资料文本切分成知识段并建立本地检索索引。随后用户在问答框中提问，软件从已学习的知识段中召回相关内容，再调用 Qwen 生成回答。

知识库页面左侧为对话区，右侧为知识库管理区。对话区显示用户问题、AI 回答、引用来源和联网搜索结果状态。管理区显示已学习文件列表、每个文件的知识段数量、文本长度、OCR 页数、学习时间和页段信息。

知识库支持当前文件学习。用户在资料阅读页选中某个 PDF 或 Markdown 后，进入知识库问答页点击“学习当前文件”，该文件会转化为知识库文档。软件会自动提取可学习文本，必要时执行 OCR。

知识库支持整门课程学习。用户点击“学习本课程”后，软件遍历当前课程资料库中的资料，逐个提取文本并加入知识库。学习完成后，知识库统计会显示文件数和知识段总数。

知识库支持页段学习。用户填写 PDF 的起始页和结束页后点击“学习页段”，软件会截取该页段，执行文字提取和 OCR，再把该页段作为独立知识来源加入知识库。页段知识源在列表中以原文件名加页码范围显示。

知识库支持子 PDF 导出。用户填写起始页和结束页后点击“导出子 PDF”，软件生成该页段对应的 PDF 副本，适合把大资料拆分成复习材料、提交材料或后续单独学习资料。

知识库支持自定义切分长度和召回段数。切分长度决定每个知识段包含的大致字符量，召回段数决定一次提问会选取多少相关知识段作为回答上下文。用户可以根据资料规模和问题复杂度调节这两个参数。

知识库支持联网搜索开关。默认回答以本地已学习资料为主。用户开启“启用联网搜索”后，软件会把问题发送到搜索服务，收集搜索摘要结果，并把搜索结果作为辅助知识加入回答上下文。回答中会展示联网搜索使用情况。

问答按钮兼具停止功能。AI 回答进行中时，提问按钮变为停止按钮，用户再次点击即可终止本次输出。该交互用于避免连续点击造成请求重叠，也方便用户中断过长回答。

知识库对话保留最近多轮消息。历史对话记录保存在当前课程 RAG 知识库中，切换课程后对话上下文随课程切换。

## 八、阿龙 Coding

阿龙 Coding 是面向编程学习和算法练习的 AI 辅助页面。用户可以选择编程语言，填写题目或需求，也可以粘贴已有代码。软件根据输入内容判断任务类型，并返回代码分析或推荐解法。

用户只填写题目时，阿龙会给出一版推荐代码，并说明核心思路、复杂度、关键边界条件和实现要点。用户同时填写题目和代码时，阿龙会重点分析代码可行性、复杂度、潜在问题、边界情况和优化方向。

页面右侧展示分析结果。结果使用 Markdown 形式呈现，包含结构化说明、代码块和建议。用户可以点击清空按钮重置输入与输出，开始下一次代码问答。

## 九、科学计算器

科学计算器页面提供仿卡西欧 fx-991MS 风格的课堂速算工具。计算器支持数字输入、四则运算、括号、百分号、倒数、平方、幂运算、阶乘、平方根、三角函数、反三角函数、常用对数、自然对数、科学计数法、圆周率、自然常数和 Ans 结果复用。

计算器支持 DEG 与 RAD 角度模式切换。三角函数会根据当前角度模式解释输入值，反三角函数会按当前角度模式输出结果。SHIFT 用于触发反函数和扩展函数，例如 sin 切换为 asin，log 切换为 10 的幂，ln 切换为 e 的幂，平方根切换为立方根。

计算器支持键盘输入。用户在计算器页面中直接按数字、运算符、回车和退格键即可操作计算器。右侧速算说明展示 Ans、角度模式和常用输入方式。

## 十、学习计划与日历

学习计划页用于管理课程复习、技术学习和项目推进任务。页面包含任务表格、状态筛选、统计卡片、今日推进列表和新增任务表单。

任务表格以阶段、类型、学习内容、状态、时间线、进度、重要程度和备注组织学习计划。用户可以把任务标记为待开始、进行中或已完成，也可以拖动进度条调整完成比例。重要程度以星级展示，用户点击星级即可调整优先级。

页面顶部提供状态筛选按钮。用户可以查看全部任务、进行中任务、待开始任务和已完成任务。统计卡片会根据当前计划数据展示总任务数、完成数量、进行中数量和整体进度。

今日推进列表根据任务状态、进度和重要程度生成，帮助用户把注意力集中到当前最值得推进的学习事项上。

日历事件与工作台共享。用户在工作台添加的重要事件会出现在日历中，学习计时也会在日历中形成学习热力标记。任务计划、日历事件和学习计时共同构成软件的时间管理模块。

## 十一、知识图谱

知识图谱页把课程资料转化为概念节点和关系网络。用户导入资料后点击生成知识图谱，软件会读取当前课程资料，提取核心概念、章节、难度、摘要、来源片段和概念关系，并生成可视化图谱。

知识图谱页面包含生成状态卡片、Neo4j 状态、搜索框、路径查询区、视图筛选区、图谱画布和节点详情栏。生成过程中，状态卡片展示当前阶段、生成进度和提示信息。图谱生成完成后，节点和边显示在画布中。

图谱画布支持拖拽移动、滚轮缩放、重置视图和节点拖拽。节点颜色和样式体现节点类型、章节、难度和掌握情况。用户点击节点后，右侧详情栏展示节点名称、说明、章节、难度、掌握度、来源片段和关联节点。

知识图谱支持视图模式切换。掌握度模式强调学习熟练程度，章节模式强调课程结构，难度模式强调复习优先级。用户也可以通过章节、难度和掌握度筛选节点，聚焦当前复习范围。

知识图谱支持搜索定位。用户输入知识点或关键词后，软件在当前图谱或 Neo4j 中搜索匹配节点，并把画布焦点移动到目标节点。

知识图谱支持展开邻居。用户选中一个节点后点击展开邻居，软件从 Neo4j 查询该节点的一度关系，并把相关概念补充到图谱中。

知识图谱支持路径查询。用户选择路径起点和终点后，软件查询两个知识点之间的关系路径，并在画布中高亮路径。该功能适合展示不同知识点之间的前置、包含、关联和应用关系。

知识图谱支持原文来源查看。节点保留与资料内容相关的片段，用户点击来源按钮即可查看对应说明，便于从图谱回到资料证据。

知识图谱支持 Mermaid 导出。用户可以把当前图谱导出为 Mermaid `graph TD` 文本，用于报告、展示文档和演示材料。

## 十二、思维导图

思维导图页以层级结构展示课程主题。它面向复习梳理，强调主题分支、层级关系和复习提示。用户可以从课程资料重新生成思维导图，也可以手动编辑节点，使导图更符合个人复习习惯。

思维导图支持经典结构、紧凑结构、按章节组织和按难度组织等展示方式。用户选择不同组织方式后，导图的层级和视觉布局会随之调整。

导图节点支持展开、收起、聚焦和缩放。用户点击分支折叠按钮可以隐藏或展开子主题；选择聚焦主题后，页面只展示该主题及相关子分支；缩放按钮用于放大或缩小导图视图。

节点详情栏展示当前选中主题的摘要、来源和复习提示。用户可以对当前节点新增子节点、编辑节点或删除节点。点击“练习当前主题”会进入自动测验流程，并以该主题为练习范围。

思维导图支持重新生成。用户导入新资料或更新课程内容后，点击重新生成即可让软件根据资料重新整理主题分支。

思维导图支持 Mermaid mindmap 导出。用户可以把当前导图导出为 Mermaid `mindmap` 文本，用于报告整理和课堂展示。

## 十三、自动测验

自动测验页把课程资料转化为练习题。测验支持单选题、多选题、判断题、填空题、简答题和概念匹配题。用户可以选择练习模式、出题范围、题目难度和各题型数量。

练习模式包括自适应练习、快速诊断、薄弱巩固、今日复习和考前冲刺。自适应练习会结合掌握度、答题历史和题目优先级排序；快速诊断用于快速覆盖核心知识点；薄弱巩固聚焦低掌握度和错题；今日复习围绕复习队列安排题目；考前冲刺混合多种题型和难度。

出题范围支持全部资料、指定知识点和导图主题。用户从知识图谱或思维导图中点击练习入口时，测验页会自动把对应知识点作为练习范围。

客观题由软件即时判分。单选题、判断题、填空题和概念匹配题会直接比对答案，多选题支持多项选择状态。答题后页面展示正确性、正确答案、解析和来源片段。

简答题支持 AI 语义评分。用户提交简答题答案后，软件调用 Qwen 对答案进行评分，返回分数、反馈和缺失要点。评分结果写入答题记录，并影响错题与掌握度统计。

错题会自动进入错题本。每条错题记录包含题目、用户答案、正确答案、解析、来源片段、错因、复习等级和下次复习时间。用户在报告页可以标记错题已复习，也可以重新练习错题对应主题。

自动测验页右侧展示今日复习队列、当前完成数量、题目列表和会话统计。用户可以切换下一题、重置题组、重新生成题目和跳转学习报告。

## 十四、学习报告与阶段回顾

学习报告页展示课程学习状态。页面顶部显示本课程练习总数、学习时长、平均正确率和待复习错题数量。趋势图展示近七天练习曲线，推荐复习区域根据薄弱知识点生成复习建议。

错题本区域自动收集答错题目。每张错题卡片展示题目、用户答案、正确答案、解析和操作按钮。用户可以标记已复习，也可以点击重新练习进入对应知识点的测验。

阶段回顾区域提供学习总结能力。用户可以选择手动总结、每天一次或每周一次。每次总结以“上次总结到本次总结”为统计窗口，首次总结从课程学习开始统计。

阶段回顾会整理学习专注与情绪情况、学习过的内容、学习资料数量、知识段覆盖情况、测验完成数量、正确率、错题数量、复习进展、活跃学习天数和下一步建议。开启摄像头状态识别后，专注分和主要情绪状态会进入回顾统计。

历史回顾列表保留最近阶段总结。用户可以通过历史回顾对比不同阶段的学习节奏、测验准确度和复习重点变化。

## 十五、状态识别、手势控制与音乐推荐

状态与音乐页整合摄像头预览、面部状态识别、手势识别、专注分展示和音乐播放。用户点击开启摄像头后，软件请求摄像头权限并显示实时画面。摄像头画面可叠加关键点显示，用户按 `X` 或点击页面按钮可以开关识别点阵。

面部状态识别会根据人脸关键点、表情特征和环境亮度判断学习状态。状态包括专注、疲劳、走神、焦虑、疑惑、轻松和离开。页面展示当前状态、状态详情、置信度和专注分。状态结果也同步给龙龙桌宠和音乐推荐模块。

手势识别用于应用内虚拟鼠标控制。右手拇指与食指捏合后，软件根据捏合点移动虚拟鼠标；左手点赞触发应用内点击；右手点赞触发向上滚动；右手倒赞触发向下滚动。手势有稳定判定和防抖机制，适合在阅读资料、浏览图谱和演示软件时进行无接触操作。

音乐推荐模块根据当前状态选择适合学习的固定歌单。专注和轻松状态倾向播放平稳音乐；疲劳、焦虑、疑惑和走神状态会切换到更舒缓或低干扰的曲目。当前歌曲播放过程中，情绪变化只更新推荐状态；歌曲自然结束后，软件会根据当前状态选择下一首。

音乐播放器支持播放、暂停、上一首、下一首、进度拖动、音量调节和本地 MP3 导入。用户导入的 MP3 会加入当前播放列表，并在状态与音乐页展示。

## 十六、龙龙桌宠助手

龙龙是 LongMindStudy 的陪伴式学习助手。主窗口右下角显示龙龙悬浮入口，桌面端还会创建独立桌宠窗口。龙龙会显示今日学习时间、学习提醒、当前状态、音乐建议和快捷操作。

用户可以拖拽龙龙改变位置。点击龙龙会触发互动反馈和语音提示。龙龙面板提供摸摸、学习提醒、聊天和礼物入口。

龙龙聊天与主窗口共享记忆。用户与龙龙对话时，软件会记录近期聊天内容和长期记忆。用户说出类似“记住……”的内容时，龙龙会把关键信息收进记忆库，并在之后的对话中参考这些记忆。

龙龙会结合课程资料、RAG 知识库、屏幕截图和近期学习状态回答问题。用户可以在龙龙聊天中提出学习问题、让龙龙解释资料、请求提醒或进行轻量陪伴式对话。

龙龙好感系统包含好感值、等级、龙币、礼物商店和已拥有礼物。学习计时会兑换龙币，用户可以使用龙币购买礼物，礼物增加好感并推动等级成长。好感数据保存在本机，主窗口和桌宠窗口同步更新。

龙龙固定语音包用于上线问候、摸摸反馈、学习提醒、AI 思考提示和回答开场。音频资源存放在 `frontend/assets/longlong-voice`，软件交互时直接播放对应固定音频。

## 十七、语音控制

语音控制是软件的自然交互入口。用户点击顶部“语音控制”按钮后，软件打开语音控制面板并进入监听状态。面板显示当前监听状态、识别文本和执行结果。

语音控制使用 Qwen ASR 将音频转为文字，再用本地控制规则识别指令。软件支持以下语音操作：

- 打开或关闭语音控制
- 导入课程资料
- 学习当前资料
- 学习本课程资料
- 生成知识图谱
- 打开上一个、下一个或指定序号的资料
- 按资料名称切换文件
- PDF 下一页、上一页或跳转到指定页
- 切换到工作台、资料阅读、知识库问答、Coding、计算器、计划、知识图谱、思维导图、测验、报告、状态与音乐页面
- 打开或关闭摄像头状态识别
- 打开龙龙聊天

语音控制对重复指令做短时间防抖。软件在持续监听期间只执行控制类内容，普通说话内容保持静默处理，使语音入口可以长期打开。

## 十八、AI 配置与外部服务

软件内置 Qwen / DashScope API Key 设置入口。用户点击界面中的 Key 按钮即可打开设置框，输入 API Key 后保存。Key 存储在 Electron 用户数据目录下，课程资料和源码仓库中只保存普通学习数据。

通用 AI 能力使用 Qwen 模型。课程问答、RAG 问答、知识图谱生成、思维导图生成、自动出题、简答题评分、代码助手、龙龙对话和语音识别都通过统一 AI 能力层调用。多模态任务可以使用视觉能力模型处理截图或图像输入。

音乐推荐使用独立的音乐推荐 Key。歌曲播放结束时，软件把当前情绪、固定歌单和最近播放记录交给 AI，由 AI 从固定歌单中选择下一首。AI 选择结果用于播放顺序，本地规则也会根据情绪匹配歌曲。

Neo4j 用于知识图谱持久化。用户运行 Neo4j 启动脚本后，软件可以检查本地图数据库状态、保存课程图谱、加载课程图谱、搜索节点、展开邻居和查找路径。连接信息保存在本机 `.local` 配置中。

OCR 使用 Tesseract。扫描版 PDF 页面会渲染为图像，再进入 OCR 识别流程。OCR 识别文本会进入当前页阅读、资料学习和 RAG 知识库。

## 十九、数据保存与学习连续性

LongMindStudy 把课程库保存在浏览器本地存储中，把学习计时、龙龙记忆、龙龙好感、Qwen Key 和语音相关配置保存在 Electron 用户数据目录中。Neo4j 数据保存在本地图数据库中。

课程库保存课程名称、课程说明、资料列表、资料路径、学习模块、RAG 知识库、对话记录、图谱数据、思维导图、测验记录、错题记录和学习报告。用户重新打开软件时，当前课程和学习进度会恢复。

资料文件来自用户本机。软件保存文件路径和导入时的内容副本，阅读、AI 分析、RAG 学习和图谱生成围绕课程资料记录展开。Markdown 在用户点击保存原文件时写回本地文件；导出的 PDF 和另存 Markdown 由用户选择保存位置。

学习计时按日期累计。计时数据用于工作台显示、日历热力、龙币兑换和学习回顾。龙龙记忆、好感和礼物数据独立保存，使陪伴体验具备连续性。

## 二十、典型使用流程

用户打开软件后进入学习工作台，先在课程选择器中新建或选择课程。随后点击顶部“导入资料”，选择本地 PDF 或 Markdown 文件。资料导入后进入资料阅读页，在左侧资料库中选择文件，中间阅读资料，右侧查看 AI 阅读分析或翻译。

用户进入知识库问答页，点击“学习当前文件”或“学习本课程”，把资料转为知识库。对于大型 PDF，用户填写起始页和结束页，只学习需要复习的页段。知识库生成后，用户在对话框中提出课程问题，软件基于已学习资料回答，并展示引用来源。

用户进入知识图谱页，点击生成知识图谱。生成完成后，用户通过图谱查看概念关系，点击节点阅读摘要和来源片段，使用搜索定位重点概念，使用路径查询理解知识点之间的联系。

用户进入思维导图页，查看课程主题层级。用户展开或折叠分支，聚焦某个主题，并在右侧查看复习提示。需要展示或报告整理时，用户导出 Mermaid mindmap 文本。

用户进入自动测验页，选择练习模式、范围和难度，生成题目后开始答题。答题结果会进入统计，错题进入错题本。用户之后在学习报告页查看正确率、薄弱知识点、错题本和阶段回顾。

学习过程中，用户可以开启状态与音乐页的摄像头识别。软件展示专注分和当前情绪状态，并根据状态推荐音乐。用户使用手势移动虚拟鼠标、点击或滚动页面，也可以打开语音控制持续监听页面切换、资料学习和 PDF 翻页指令。

用户在右下角与龙龙互动，查看学习时间、接收提醒、进行聊天、积累龙币和赠送礼物。龙龙把学习计时、状态识别、音乐建议和 AI 对话串联成更轻松的学习陪伴体验。

## 二十一、运行方式

用户在项目根目录安装依赖：

```bash
npm install
```

用户启动桌面端软件：

```bash
npm start
```

用户启动或检查 Neo4j：

```bash
npm run neo4j:start
npm run neo4j:check
```

用户准备龙龙固定语音素材：

```bash
npm run prepare:voice
```

用户在软件中点击 Key 按钮配置 Qwen / DashScope API Key。用户也可以在终端中设置环境变量：

```powershell
$env:DASHSCOPE_API_KEY="your_api_key_here"
npm.cmd start
```

音乐推荐使用独立环境变量：

```powershell
$env:Music_Recommand_Key="your_music_recommend_api_key_here"
npm.cmd start
```

## 二十二、软件整体价值

LongMindStudy 把课程资料从“静态文件”转化为“可交互学习系统”。资料阅读负责输入，AI 阅读和 RAG 问答负责理解，知识图谱和思维导图负责结构化，自动测验和错题本负责巩固，学习报告负责回顾，状态识别、音乐推荐、语音控制和龙龙桌宠负责提升学习体验。

软件的整体设计围绕学习闭环展开：导入资料、理解资料、建立知识库、形成知识结构、练习题目、记录错题、复盘报告、调节状态。用户在同一套桌面工作台中持续完成这些操作，学习内容、学习行为和学习状态形成可积累的数据资产。
